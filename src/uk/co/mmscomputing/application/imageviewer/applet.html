mmsc - multipage imageviewer : test program for twain and sane wrapper and various imageio readers and writers (gif, sff, tiff)

This is an applet that needs access to a native library (jtwain.dll or libjsane.so). The applet will only be able to access this library if you accept the security certificate it was signed with. But be aware: If you accept the certificate the applet has full access to your computer!
